# Package index

## DNA sequences

- [`ft_seqs()`](https://fernphy.github.io/ftolr/reference/ft_seqs.md) :
  Load the DNA sequences used to build the Fern Tree of Life (FTOL)

## Phylogeny

- [`ft_tree()`](https://fernphy.github.io/ftolr/reference/ft_tree.md) :
  Load the Fern Tree of Life (FTOL)

## Versioning

- [`ft_data_ver()`](https://fernphy.github.io/ftolr/reference/ft_data_ver.md)
  : Current Fern Tree of Life (FTOL) version and related metadata

## Data

Datasets included in this package

### Accessions

- [`accessions_long`](https://fernphy.github.io/ftolr/reference/accessions_long.md)
  : GenBank accessions used in the Fern Tree of Life (FTOL), long format
- [`accessions_wide`](https://fernphy.github.io/ftolr/reference/accessions_wide.md)
  : GenBank accessions used in the Fern Tree of Life (FTOL), wide format

### Alignments

- [`plastome_alignment`](https://fernphy.github.io/ftolr/reference/plastome_alignment.md)
  : Aligned plastome DNA sequences used to build Fern Tree of Life
  (FTOL)
- [`plastome_parts`](https://fernphy.github.io/ftolr/reference/plastome_parts.md)
  : Start and end positions of loci in plastome DNA sequence alignment
- [`sanger_alignment`](https://fernphy.github.io/ftolr/reference/sanger_alignment.md)
  : Aligned (mostly) Sanger DNA sequences used to build Fern Tree of
  Life (FTOL)
- [`sanger_parts`](https://fernphy.github.io/ftolr/reference/sanger_parts.md)
  : Start and end positions of loci in Sanger DNA sequence alignment

### Fossils

- [`con_fossils`](https://fernphy.github.io/ftolr/reference/con_fossils.md)
  [`ml_fossils`](https://fernphy.github.io/ftolr/reference/con_fossils.md)
  : Fossil calibration points used for dating Fern Tree of Life (FTOL)

### Taxonomy

- [`ftol_ferns`](https://fernphy.github.io/ftolr/reference/ftol_ferns.md)
  : Fern Tree of Life (FTOL) fern species
- [`ftol_match_results`](https://fernphy.github.io/ftolr/reference/ftol_match_results.md)
  : Fern Tree of Life (FTOL) taxonomic name matching results
- [`ftol_taxonomy`](https://fernphy.github.io/ftolr/reference/ftol_taxonomy.md)
  : Fern Tree of Life (FTOL) taxonomic data

### Trees

- [`backbone_tree`](https://fernphy.github.io/ftolr/reference/backbone_tree.md)
  : Fern Tree of Life (FTOL) backbone phylogeny
- [`ftol_con_dated_tree`](https://fernphy.github.io/ftolr/reference/ftol_con_dated_tree.md)
  : Fern Tree of Life (FTOL) dated consensus phylogeny
- [`ftol_con_tree`](https://fernphy.github.io/ftolr/reference/ftol_con_tree.md)
  : Fern Tree of Life (FTOL) consensus phylogeny
- [`ftol_ml_dated_tree`](https://fernphy.github.io/ftolr/reference/ftol_ml_dated_tree.md)
  : Fern Tree of Life (FTOL) dated maximum-likelihood phylogeny
- [`ftol_ml_tree`](https://fernphy.github.io/ftolr/reference/ftol_ml_tree.md)
  : Fern Tree of Life (FTOL) maximum-likelihood phylogeny
