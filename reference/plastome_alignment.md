# Aligned plastome DNA sequences used to build Fern Tree of Life (FTOL)

Concatenation of 79 loci from whole fern plastomes, including maximal
sampling of available species on GenBank.

## Usage

``` r
plastome_alignment
```

## Format

Matrix of class "DNAbin"

## Details

Includes 504 species and 75,413 bases.

For details on methods used to assemble alignment, see Nitta et al.
2022.
