# Fern Tree of Life (FTOL) backbone phylogeny

Majority-rule extended consensus phylogenetic tree inferred from whole
fern plastomes by maximum-likelihood, including maximal sampling of
available species on GenBank.

## Usage

``` r
backbone_tree
```

## Format

List of class "phylo"

## Details

Includes 504 tips and 503 internal nodes. Branch lengths are in units of
estimated genetic change per site (not ultrametric). Node labels are
bootstrap support (%).

For details on methods used to infer the tree, see Nitta et al. 2022.
