# Aligned (mostly) Sanger DNA sequences used to build Fern Tree of Life (FTOL)

Concatenation of 7 loci primarily obtained from Sanger sequencing,
including maximal sampling of available species on GenBank.

## Usage

``` r
sanger_alignment
```

## Format

Matrix of class "DNAbin"

## Details

Includes 5,611 species and 12,648 bases.

For details on methods used to assemble alignment, see Nitta et al.
2022.
